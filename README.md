> A reproducible framework for financial market risk assessment using hybrid Quantum-Classical Neural Networks.

## 📌 Overview

This repository implements a 4-qubit hybrid quantum-classical neural network (QNN) for financial risk prediction on S&P 500, FTSE 100, EUR/USD, and GBP/USD data.

**Status:** Work in progress. Results will be added upon completion.
